# Course Management System

A Spring Boot application for managing courses, instructors, and students with role-based authentication.

## Features
- Role-based access control (Admin, Instructor, Student)
- Manage courses, students, and instructors
- Secure authentication using Spring Security + JWT
- RESTful API with ModelMapper DTO mapping
- H2 / MySQL database support

## Tech Stack
- Java 17  
- Spring Boot 3  
- Spring Security  
- JPA / Hibernate  
- ModelMapper  
- Maven

git clone https://github.com/ToqaHatem1/CourseManagementSystem.git
cd CourseManagementSystem
